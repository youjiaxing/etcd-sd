# etcd-service-discovery
基于 etcd 实现的轻量级服务注册发现库

## 功能特性
✅ 服务注册与自动续约  
✅ 服务发现与实时监听  
✅ 多命名空间支持  
✅ 异常重试机制

## 快速使用
```go
go get github.com/youjiaxing/etcd-sd@latest
```
